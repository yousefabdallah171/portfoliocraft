Modern multipurpose WordPress theme with Elementor and WooCommerce integration.

== Description ==

portfoliocraft is a modern, multipurpose WordPress theme designed for businesses, portfolios, and e-commerce websites. Built with Elementor page builder integration, WooCommerce compatibility, and a powerful theme options panel.

= Key Features =

* Elementor Page Builder Integration
* WooCommerce Ready
* Dark Mode Support
* Responsive Design
* Custom Post Types
* Extensive Theme Options
* Contact Form 7 Integration
* SEO Optimized
* Translation Ready
* RTL Support

= Required Plugins =

* Elementor (from WordPress repository)
* portfoliocraft Core (included)
* WooCommerce
* Contact Form 7

== Installation ==

1. Upload the theme files to `/wp-content/themes/portfoliocraft`
2. Activate the theme through the 'Themes' screen in WordPress
3. Install and activate required plugins
4. Use the theme customizer to set up your site
